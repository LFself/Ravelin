Detailed Description

Defines a sliding joint.

Constructor & Destructor Documentation

PRISMATICJOINT::PRISMATICJOINT ( )

Initializes the joint.

The axis of rotation is set to [0 0 0]. The inboard and outboard links are set to NULL.

void PRISMATICJOINT::set_axis ( const VECTOR3 &  axis)

Sets the axis of translation for this joint (MUST BE CALLED AFTER set_location(.))

The local axis for this joint does not take the orientation of the inboard link into account; thus, if the orientation of the inboard link changes, then the local axis remains constant.
